# What is DSA

DSA stands for Data Structures and Algorithms. It's the Computer Science field where you learn how to write effective (that solves the problem) and efficient (that is fast) code. 

Therefore, DSA is divided into two main branches: 

- DS: What are the ways to store data?
- A: What are the procedures we can make? 

Both DS and A have well known concepts, and you can't (or at least it's difficult to avoid) explain a DS without applying any A to it. So throughout the lectures, you'll see both.
